HOW TO USE
================================================================================

FOR GOOGLE COLAB DEPLOYMENT:
-----------------------------
1. Go to: https://colab.research.google.com/
2. Upload: cloud_deployment/colab_quickstart.ipynb
3. Enable GPU: Runtime > Change runtime type > T4 GPU
4. Follow the notebook steps:
   - It will ask you to upload the 7 files from cloud_deployment/
5. Run all cells
6. Get your public Gradio link!

FOR LOCAL USE:
--------------
All files are still in the main directory:
1. Install: pip install -r requirements.txt
2. Run: python app.py
   (Note: Will need GPU or use simplified model for Mistral 7B)
